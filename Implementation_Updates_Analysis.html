Update 1: State Dimension Expansion (39 → 42)

✓ Constraint Features Added

The policy was blind to advertiser constraints. Now it sees them as normalized features, enabling constraint-aware decision-making instead of post-hoc guardrails.

📌 target_cpa_norm (Index 39)
Normalized target cost-per-acquisition. Calculation: log1p(target_cpa) / log1p(1000). Helps SAC avoid overspending per conversion.
📌 min_roas_norm (Index 40)
Normalized minimum return-on-ad-spend. Calculation: min_roas / 10.0. SAC learns to maintain minimum profitability threshold.
📌 daily_budget_limit_norm (Index 41)
Normalized daily budget ceiling. Calculation: log1p(limit) / log1p(MAX_DAILY_BUDGET). Prevents budget-aggressive decisions.
